# JPDB Project Management System

## Description

This project is a Project Management System developed using HTML, Bootstrap, JavaScript, jQuery, and JsonPowerDB (JPDB).

The application allows users to create, retrieve, update, and manage project records stored in JsonPowerDB. Project ID acts as the primary key and is used to uniquely identify each project record.

The system automatically checks whether a Project ID already exists in the database. If the record exists, the form is populated with the stored data and allows updating. If the record does not exist, the user can create a new record and save it to the database.

## Scope of Functionalities

### New Project Entry
- Enter a new Project ID.
- Create and save new project records.

### Existing Project Retrieval
- Fetch project information using Project ID.
- Auto-populate form fields.

### Project Update
- Modify existing project details.
- Update records directly in JsonPowerDB.

### Form Validation
- Prevent empty field submissions.
- Ensure data consistency.

### Form Reset
- Clear form fields.
- Return form to initial state.
